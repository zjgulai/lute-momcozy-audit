💡 So-What · 必须说出来的一句话
流量是优势 · 也是陷阱。
(a) 优势：Momcozy 已建立母婴 DTC 罕见的"饱和攻击"流量结构 (45% paid + 22.6K KOL) · 行业其他玩家短期复制不了。
(b) 陷阱：自然搜索仅 5.80% · 说明品牌缺自然势能 · 流量高度依赖持续买量 · CAC 风险大。
(c) 决策含义：M2 修漏斗的 ROI 远高于继续加买量 · 因为流量已超规模 · 钱没留下来 · 每 1% CVR 提升 = ~$18M 年增收（按 547.8K 订单基数）。
§ 2.6 · 三大资产归因 · 凭什么 + 可持续吗 + 如何放大

3 个超优 KPI · 都不是侥幸。咨询师必问"凭什么这么好 + 会被竞品复制吗 + 怎么放大"。

  • 套装策略 ~25%：配件包 Bundle $29-$49 · 提升结账件数

资产 #2 · 用户满意

28.8%

战略含义：M2 优先级 = 修 PDP 加购按钮 / 性能 · 不要碰 Checkout（碰反而可能破坏现有 28.8% 优势）
💎 三资产合一 · 战略叙事
Momcozy 是"已经做对了 3 件事的品牌" · 不要为修 CVR 牺牲任何一个
对 CMO / CEO 的核心建议：
① 不能降价拉 CVR · 因为 AOV $119 是品牌定位资产 · 降价 = 主动侵蚀核心优势

③ 不能优化 Checkout · Cart Abandonment 28.8% 已优秀 · 优化重心应该是访客 → 加购这一节（漏斗第一段损 96.3%）

§ 7 · 神秘脚本溯源

        
2 个未知第三方 · 已查清 · 全部 SAFE

        
Sisyphus 5 步通用溯源法：拉源码 → grep theme.liquid → Shopify Customer Events 后台 → DDG Tracker Radar → 同伴站点验证。

      


      

        

          

            ✓ SAFE 保留
            c.albss.com
          

          

            真身：AppLovin / Axon（NASDAQ:APP · e-commerce 归因 + 再营销 pixel）

            event_key：a9f70cb5-... · Shopify Pixel id 41976006 · status active
          

          
    
            
  1. curl loader.iife.js 末尾明文 import res4.applovin.com/.../bs.iife.js + hs.iife.js
    2. 
            
  2. grep theme.liquid 找 window.ALBSS = { event_key: '...' }
    3. 
            
  3. Shopify Customer Events 后台找 "Axon Shopify Pixel" CUSTOM 类型
    4. 
            
  4. DDG Tracker Radar：3 站点 · 0 cookie · 0 fingerprinting · 已知归因 pixel
    5. 
            
  5. 同伴站点 draxe.com / speechify.com / anker.com 都装这个 · 高端 DTC 标配
    6. 
          

          

            建议：保留 + 加 <link rel="dns-prefetch" href="//res4.applovin.com"> 节省 80-120ms LCP。
          

        


        

          

            ✓ SAFE 保留
            fsh9wdj.com
          

          

            真身：Everflow（affiliate marketing platform · partner attribution）

            与 c.albss.com 同模式 · Momcozy 主动安装的归因 SaaS
          

          
    
            
  1. 解码 fsh9wdj.com loader → 跳 everflow.io/sdk
    2. 
            
  2. grep theme.liquid 找 _ef.identify(...) 调用点
    3. 
            
  3. Shopify Apps 搜 "Everflow" 看是否官方安装
    4. 
            
  4. DDG Tracker Radar：affiliate 行业标记 · 合规追踪
    5. 
            
  5. 同行 GymShark / Fashion Nova 等 affiliate 重的站都用
    6. 
          

          

            建议：保留 · 删了 = Momcozy affiliate 销售归因数据全断。
          

        

      


      

        
⚠ Anti-pattern 警告

        

          直接删第三方 pixel = 砍归因数据 = 广告 ROI 数据全断。
          AppLovin Axon / Everflow / Meta Pixel / Google Analytics 这类是 Momcozy 主动安装的归因 SaaS · 看似"性能负担"，实际是营销 ROI 量化的眼睛。
          删之前必须：(1) 路特核合约是否在用 · (2) 联 marketing 团队确认 ROI · (3) 留 7 天回滚窗口。
